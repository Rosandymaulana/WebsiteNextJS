Topic : Keyboard

Apa Itu

Keyboard ?

Keyboard adalah sebuah perangkat keras (hardware) pada komputer yang berfungsi sebagai alat untuk input data yang berupa huruf, angka dan simbol. Atau definisi keyboard yaitu suatu perangkat keras pada komputer yang berbentuk papan dan memiliki berbagai macam tombol yang dimana tombol tersebut fungsinya berbeda tergantung pada penekanannya yang bisa menghasilkan proses yang sesuai dengan keinginan penggunanya.

Apa Keyboard yang kami kenalkan ?

Keyboard memiliki 4 bagian utama yaitu: Typewriter Key, Numeric Key, Function Key, Special Function Key. Jenis keyboard dibagi menjadi dua bagian yaitu berdasarkan tombol dan bentuk. Berikut contoh macam macam Keyboard berdasarkan tombol yang dapat dibedakan menjadi enam, yaitu Keyboard Qwerty, Keyboard Maltron,Keyboard Chord, Keyboard Drovak, Keyboard Alphabetic, dan Keyboard Numeric.
