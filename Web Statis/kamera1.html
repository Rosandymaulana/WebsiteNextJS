Canon EOS 200D Rebel SL2 ?

Canon EOS 200D Rebel SL2 adalah kamera DSLR level pemula yang menampilkan sebagian teknologi terbaru dalam seri EOS, seperti monitor LCD Vari-angle dan Dual Pixel CMOS AF.

Canon EOS 200D Rebel SL2

Type : Digital, single-lens reflex, AF/AE camera with built-in flash
Recording media
: SD/SDHC*/SDXC* memory cards, UHS-I cards supported
Image sensor size : Approx. 22.3 x 14.9 mm.
Compatible lenses : Canon EF lenses (including EF-S lenses), Excluding EF-M lenses (35mm-equivalent angle of view. 1.6x the focal length indicated.)
Lens mount : Canon EF mount
